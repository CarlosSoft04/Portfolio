Alguns Projetos

SmartHome

Em equipe, desenvolvemos uma solução de automação residencial, permitindo que os usuários controlem remotamente até 4 lâmpadas através de seus smartphones. Utilizamos React, Node.js, MQTT e bancos de dados para criar uma experiência intuitiva e de fácil controle, simulando um ambiente de "Casa Inteligente". Esse projeto foi uma excelente oportunidade de aplicar conceitos de IoT e integração de sistemas. Link do projeto

SchoolPayroll

Sistema de folha de pagamento para gerenciar professores de uma escola. A aplicação utiliza um banco de dados em memória (H2), acessado via JPA, e é exposta por meio de uma camada Spring MVC + Spring HATEOAS para acesso pela web Link do projeto

ColorGuess

Color Guess é um jogo interativo e divertido onde os jogadores precisam adivinhar o código hexadecimal da cor exibida na tela. O jogo testa suas habilidades de percepção de cores e oferece uma experiência simples, mas desafiadora. Quanto mais você acerta, mais difícil o jogo fica! Link para o projeto

Bot para Discord
